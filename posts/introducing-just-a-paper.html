Introducing Just a Paper – Minimalist Writing App

July 3, 2025

The app name is Just a Paper. As the name suggests, It's just like paper.

It has only four features: change font style, change font size, adjust width, and change line height. Here's how to install it:

  • Install from Github page (macOS only)
  • Run the command below before opening the app
  • $ sudo xattr -cr "/Applications/Just a Paper.app"

You need to run the command because it's a test app. Without the command, you'll get a warning. The app is open source. You can download and customize it.
